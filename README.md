# Machine Learning Laboratory – Classification, Clustering & Dimensionality Reduction

This repository consolidates end-to-end implementations of supervised and unsupervised learning workflows, including classification benchmarking, clustering performance evaluation, and dimensionality reduction via PCA.

Each task strictly follows academic lab specifications and is structured for reproducibility, comparative reporting, and portfolio readiness.

---

## Lab Portfolio

### **01 – Logistic Regression vs Multinomial Naive Bayes**

* Binary / multi-class classification
* Same dataset used for both models
* Comparative performance reporting

### **02 – SVM vs Random Forest**

* Dataset distribution visualization
* Metrics: Accuracy, Precision, Recall, F1-Score
* Hyperparameter impact assessment

### **03 – K-Means vs Gaussian Mixture Model (GMM)**

* Unsupervised clustering comparison
* Evaluation via inertia, silhouette score, likelihood

### **04 – Principal Component Analysis (PCA)**

* Dimensionality reduction
* Variance retention curve
* Performance impact on downstream classifier
